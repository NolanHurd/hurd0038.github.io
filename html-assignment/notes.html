HTML Tags - Examples

Paragraphs
To add a paragraph of text on the web page, use the paragraph tag. Also worth noting: the tag will automatically add a space both before and after the paragraph.
Blockquote
To add a quote from another source onto your web page, use the Blockquote tag; this will add a large, stand-alone quote in its own paragraph.
Highlighting Text
To highlight a portion of the text on the webpage, use the "mark" tag; will present the text in a bright colour that catches the reader's attention.
Subscript Text
To define text as subscript (Example: CO2), use the "sub" tag in the code.
Superscript Text
To define text as superscript (Example: "I'm celebrating my 21st birthday!"), use the "sup" tag in the code
